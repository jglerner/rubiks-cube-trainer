A modern desktop application for practicing Rubik's Cube with timer, statistics, and 281 trillion unique scrambles.

## Features

- Interactive 3D Cube with drag-to-rotate
- Smart Scrambler (2-25 move sequences with unique IDs)
- Professional Timer (Space to start/stop, R to reset)
- Advanced Statistics (ao5, ao12, ao100, DNF tracking)
- Step-by-Step Mode for learning
- Persistent History
- Draggable UI windows
- Visual 2D scramble display

- R: Reset timer
- Mouse Drag: Rotate cube
